# BattleshipJS
A simple battleship game built using Vanilla JS + HTML + CSS

Live demo: https://battleship.miguelhiguera.dev/

## Features
- Player vs. AI
- Smart AI (hits adjacent tiles after hitting a ship using a depth-first search-like algorithm)

## Build steps
- Clone repository
